# DataHaul MVP - Verified Farm Input Marketplace

## Target Market
- Maize and vegetable farmers (1-5 acres)
- Central and Greater Accra regions, Ghana
- Currently buying from local agro-dealers

## Core Value Proposition
Verified farm input marketplace with transparent pricing and trust guarantees

## MVP Features
1. Phone number + OTP authentication
2. Supplier verification system (admin-approved)
3. Input catalog with price comparison
4. Simple ordering (pay-on-delivery/pickup)
5. Order tracking and updates

## Success Metrics
- % farmers placing first order
- Repeat purchases in same season
- Transaction fee acceptance rate

## Tech Stack
- Frontend: React Native/Expo (mobile-first PWA)
- Backend: Django REST Framework (existing)
- Auth: Phone + OTP
- Database: PostgreSQL
- Hosting: Render.com + Vercel
